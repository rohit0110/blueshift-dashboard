# Tạo một SPL Token

![Mint an SPL Token Challenge](/graphics/challenge-banners/typescript-mint-an-spl-token.png)

Chương trình SPL token, và cụ thể là tạo một số token bằng TypeScript, nên là điểm khởi đầu đầu tiên trong hành trình của bạn với tư cách là một Solana developer.

Nếu bạn mới làm quen với chương trình SPL Token, hãy theo dõi [khóa học này](/vi/courses/spl-token-with-web3js) trước!

Trong thử thách này, chúng ta sẽ triển khai bốn instruction đơn giản:

- **Tạo một Mint**: Tạo `Mint Account` bằng cách sử dụng raw instruction hoặc abstracted instruction mà SDK có sẵn.
- **Khởi tạo Mint**: Khởi tạo `Mint Account` đã tạo bằng cách sử dụng raw instruction hoặc abstracted instruction mà SDK có. *Mint phải có decimal là 6, với mint authority được đặt thành wallet đang chạy code, và không có freeze authority*.
- **Tạo Associated Token Account**: Khởi tạo một `Associated Token Account` bằng cách sử dụng raw instruction hoặc abstracted instruction mà SDK có.
- **Đúc 21 triệu Token**: đúc 21 triệu (21,000,000) token mới tạo vào `Associated Token Account` ở trên.

**Lưu ý**: Nếu bạn chưa quen với cách sử dụng chương trình SPL-Token với TypeScript, bạn nên bắt đầu bằng cách đọc [SPL Token với Web3.js](/vi/courses/spl-token-with-web3js) để làm quen với các SDK mà chúng ta sẽ cần trong thử thách này.
