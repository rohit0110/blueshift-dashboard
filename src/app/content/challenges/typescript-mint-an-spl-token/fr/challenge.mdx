# Mint un Jeton SPL

![Mint an SPL Token Challenge](/graphics/challenge-banners/typescript-mint-an-spl-token.png)

Le Programme de Jetons SPL, et plus particulièrement la création de jetons à l'aide de TypeScript, devrait être le premier point de départ de votre parcours en tant que développeur Solana.

Si vous êtes nouveau dans le Programme de Jetons SPL, suivez [ce cours](/en/courses/spl-token-with-web3js) avant !

Dans ce challenge, nous allons implémenter quatre instructions simples :

- **Créer un Mint**: Crée un `Mint Account` à l'aide d'une instruction brute ou d'une instruction abstraite fournie par le SDK.
- **Initialiser Mint**: Initialise le `Mint Account` créé à l'aide d'une instruction brute ou d'une instruction abstraite fournie par le SDK. *Le mint doit avoir 6 décimales, l'autorité de mint (mint authority) doit être le portefeuille qui exécute le code et il n'y a aucune autorité de gel (freeze authority)*.
- **Créer un Compte de Jeton Associé**: Crée et initialise un `Associated Token Account` à l'aide d'une instruction brute ou d'une instruction abstraite fournie par le SDK.
- **Mint 21 millions de Jetons**: Mint 21 millions (21 000 000) de jetons nouvellement créés au `Associated Token Account` nouvellement créé.

**Remarque** : Si vous ne savez pas comment utiliser le programme SPL-Token avec TypeScript, commencez par lire le cours [SPL Token avec Web3.js](/en/courses/spl-token-with-web3js) afin de vous familiariser avec les SDKs dont nous aurons besoin dans ce challenge.
